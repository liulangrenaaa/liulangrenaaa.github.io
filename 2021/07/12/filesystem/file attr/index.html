file attributes 概念

除了控制用户和组读取，写和执行权限的文件模式位之外，几个文件系统支持文件属性，
可以进一步自定义允许的文件操作。

cp rsync 等操作不会保留 file attributes。
只有部分 filesystem 支持这样的属性，常见的 ext2 ext3 ext4 都是支持的。
